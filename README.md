# ML Roadmap

A structured machine learning learning repository covering:

- Data preparation and exploratory analysis
- Classical supervised learning
- Model evaluation and error analysis
- Neural network foundations
- Model serving and monitoring
- Retrieval evaluation

## Repository Structure

- `src/` — reusable application and pipeline code
- `notebooks/` — exploration and isolated experiments
- `notes/` — concept notes and decision logs
- `tests/` — automated tests
- `configs/` — pipeline and model configuration
- `data/raw/` — original datasets
- `data/processed/` — transformed datasets
- `reports/` — evaluation reports and project artifacts
